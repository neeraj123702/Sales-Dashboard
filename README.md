# Sales-Dashboard
📊 Sales Performance Dashboard (Excel)
📌 Project Overview

The Sales Performance Dashboard is an interactive data analysis project developed using Microsoft Excel. The purpose of this project is to analyze sales data and convert raw transactional records into meaningful business insights through visual reporting and automated calculations.

This dashboard helps in monitoring revenue, profit, regional performance, and product-wise sales trends in a structured and user-friendly format.

🎯 Project Objective

To track overall sales performance

To analyze profit and revenue trends

To compare regional sales performance

To identify top-performing products

To create a visually interactive business dashboard

To support data-driven decision making

📊 Dashboard Features

Interactive dashboard layout

KPI cards for:

Total Sales

Total Profit

Total Orders

Region-wise sales analysis

Product category performance analysis

Monthly sales trend visualization

Dynamic filtering using slicers

Pivot Tables and Pivot Charts integration

Automated calculations using Excel formulas

🔢 Excel Techniques Used

Pivot Tables

Pivot Charts

SUM & IF functions

Percentage calculations

Data Validation

Conditional Formatting

Data Cleaning and Structuring

📈 Key Insights Generated

Identification of highest revenue-generating regions

Top-performing products and categories

Monthly growth or decline in sales

Profitability analysis

Business performance comparison
